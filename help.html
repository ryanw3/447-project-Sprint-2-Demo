How to use this webpage

The Map

The map can be interacted with by clicking and draging the cursor around, If you click on a state that state's prison data will be displayed on the left of the map.

Selecting a date

To change the date you are looking at use the box next to the word "Start Date" in this box you can type out a date in the mm/dd/yyyy format or you can use the calender icon on the right side of the box to select a date from a calender view.

Navigation Bar

On the Black bar running across your screen you may see two to three options on it such as "Home" "About" and "Help", by clicking on any of these options you will be taken to that page.

"Home" is the main webpage here you can see the covid data for any given day and also login in.

"About" is the page where we've listed a few things about this webpage along with some information about us, the lovely people who worked hard to make it.

"Help" Is the page you are currently on so dont be scared if you dont currently see this option in the bar, This page is used to help break down how to use and interact with this webpage, if you ever get stuck trying to use this page this is the place to find answers.
